## **_Description of the project_**
This project is an employee tracker, CRUD application, using Nodejs. Questions are prompted in the CLI and the user's answers will modify the database.

## **_Technologies used_**
Project is created using:
* JavaScript
* Nodejs
* Expressjs
* MySQL database
* MySQL2
* console.table
